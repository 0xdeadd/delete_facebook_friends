Facebook Friend Deletion Tool

Delete all your Facebook friends with a simple script

This tool provides a script that you can run in your browser's console to mass unfriend people on Facebook. The script works by automatically clicking the "More" buttons on friend cards, selecting "Unfriend" from the menu, and confirming the action.

⚠️ Warning

This action cannot be undone. Once you unfriend someone, you'll need to send them a new friend request if you want to reconnect.

Use this tool at your own risk. This is not affiliated with Facebook in any way.

How to Use

  1. Log in to your Facebook account
  2. Go to https://www.facebook.com/friends/list
  3. Open your browser's developer tools:
    • Chrome/Edge: Press F12 or right-click and select "Inspect"
    • Firefox: Press F12 or right-click and select "Inspect Element"
    • Safari: Enable developer menu in Preferences > Advanced, then right-click and select "Inspect Element"
  4. Click on the "Console" tab in developer tools
  5. Copy the script below
  6. Paste it into the console and press Enter
  7. In the console, type deleteThisMany(400) to unfriend 400 friends (change this number as needed)
  8. Wait for the script to finish running
